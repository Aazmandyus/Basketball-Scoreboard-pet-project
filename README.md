# Basketball Scoreboard Project

A simple basketball scoreboard built with HTML, CSS, and JavaScript.

This project was created as part of my learning journey in full-stack development. The goal was to practise basic DOM manipulation, button click events, styling with CSS, and simple JavaScript functions.

## Demo

The scoreboard allows users to add points to two teams: HOME and AWAY.

Each team has three buttons:

+1 point
+2 points
+3 points

There is also a reset button to set both scores back to zero.

## Features
Add 1, 2, or 3 points to the HOME team
Add 1, 2, or 3 points to the AWAY team
Reset both scores to 0
Scoreboard-style design
Custom fonts and colors
Responsive basic layout using Flexbox
Technologies Used
HTML
CSS
JavaScript

## What I Learned
While building this project, I practised:
- Creating a basic HTML structure
- Styling elements with CSS
- Using Flexbox for layout
- Selecting HTML elements with document.getElementById()
- Updating text on the page with textContent
- Creating JavaScript functions
- Using onclick to run functions when buttons are clicked
- Debugging with console.log()
